# Blog Design System

## Aesthetic
- Style: Editorial luxury — think NYT Magazine meets Kinfolk
- Fonts: Playfair Display for headings, DM Serif Text for body
- Colors: Cream (#faf8f4), near-black (#1a1612), gold accent (#c9a84c)
- No generic AI aesthetics (no purple gradients, no Inter font)

## Components
- Always include a reading progress bar
- Pull quotes in italic serif, left-bordered
- Images with generous whitespace
